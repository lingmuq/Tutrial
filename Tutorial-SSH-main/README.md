# SSH接続および操作について 

## サーバー計算機へのSSH接続の注意事項
### 1. サーバーは実験段階で利用してください
大規模な計算を伴わないプログラム作成・修正段階であれば，ローカル計算機を利用．

### 2. サーバーを利用する場合は，Slackチャンネル#schedule_gpu で宣言してください
例：6/24 18:00~22:00まで「exp1-gakusei」を使います．

### 3. データは必ず作業ディレクトリ”/mnt/Data/Lab/<名前>”に保存してください．
自身の作業ディレクトリ以外に保存してあるデータは削除します．

### 4. データセットはサーバー上の別ディレクトリに置くので堀田に声をかけてください
ディスク容量の削減のため，作業ディレクトリに置かないでください．（ただし，自前データセットの場合は除く．）

### 5. 必ずPython環境は自分のanaconda仮想環境で行ってください
Anaconda (base) 環境を変更しないように．<br>
Anaconda環境は，`苗字( or 名前)`や`苗字( or 名前)-(プロジェクト名)`にしてください．<br>
例えば，`hotta`や`hotta-IET2024`．

### 6. 不要になったAnaconda環境は削除してください
ディスク容量の削減のため．
